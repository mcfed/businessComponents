# Ellipsis 组件

> 文本超出截断，并用 tooltip 做截断内容显示

## Usage

```
<Ellipsis text={text} tooltiptext={tooltipText}></Ellipsis>

```

## Option

| 参数        | 说明                                                                       | 类型    |
| ----------- | -------------------------------------------------------------------------- | ------- |
| text        | 需要显示的 text 文本                                                       |  string | 必填 |
| tooltipText | 扩展显示的 tooltipText 文本，在出现 text 与 tooltipText 不一样的场景时使用 | String  |
